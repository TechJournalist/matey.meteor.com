    Tell Meteor that whenever anything changes, rerun the function which populates the ChessBoard.JS board with the most recent one in our subscription. Critically, the first line establishes a dependency on the count of the # of gamestates, causing this function to be rerun whenever that count changes.

      Deps.autorun (c)->
    state_count = GameStates.find().count()

    unless c.firstRun
      console.debug("Reactive data change detected")
      console.debug("Board count is #{state_count-1}")

      gameState = _.last(GameStates.find().fetch())
      if gameState
        Board.render(gameState)
